## Descripción

Sistema Integral de Gestión de Autorizaciones para el Tratamiento de Datos Personales

## Instalación y Uso

1. Clona el repositorio:
    ```bash
    git clone https://github.com/usuario/repo.git
    ```

2. Instala las dependencias:
    ```bash
    npm install
    ```

3. Corre la aplicación:
    ```bash
    ng serve
    ```
